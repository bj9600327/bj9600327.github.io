网管工作日志

2021年8月24日

新的笔记本交付使用前

  1. 登记Mac地址：在有网的电脑访问无线控制器192.168.130.3；填写新电脑的Mac地址。如果是手机要连接Office-Network的wifi，注意是否为随机mac，如果是就要改为私有。批量录入完mac地址后记得到右上角按保存
  2. 连接Office-Network的wifi，是否查找打印机选是。分区，512G的硬盘只分C、D区
  3. 先做win7激活、装设计软件、装独立显卡的驱动（如有）和SAP，因为这很依赖运气
  4. 安装office2013、pdf阅读器、微信、QQ和打印机驱动；如果是海外，装上钉钉电脑版
  5. 配置打印机、加域、开放管理员权限
  6. 新电脑的哪些操作要在登入使用者的域账户后进行

  • ！！！SAP！！！

  • 如果给财务用，还需要改：

    • 注册表
    • 兼容性
    • 金算盘给外国用时：要用加密外设测试加密狗，但可省下编辑注册表计算机\HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\gasoft步骤

  • 如果安装时没有用NT优化调整，还需要：

    • 去掉搜索框
    • 更改用户账户控制
    • 个性化-主题-显示“此电脑”图标

  • 通达OA精灵

为坏掉的电脑重装系统注意事项

  1. 进PE或安全模式取出数据
  2. 进PE检查硬盘情况，如果硬盘坏了就要换硬盘
  3. PE重装系统，视电脑配置装win7旗舰版或win10专业版
  4. 破解系统。win7试试用KMS10.zip破解
  5. 加域，加入管理员组，财务电脑装金算盘
  6. 储运部的电脑不用加域

2021年8月26日

开放管理员权限

  1. 我的电脑->右键->管理->本地用户和组->组-> Administrators->添加->高级
  2. 确保“查找位置”一栏是sincerity.com。如果不是
    • 检查网络
    • 检查DNS
  3. 立即查找->把该添加的账户添加到管理员组

将一台电脑加入域主机

我的电脑->右键->属性->高级系统设置->计算机名->更改->隶属于->域->sincerity.com->确定

  1. 如果报错“无法与…域控制器（AD DC）连接”
  • 先确保连进了公司网络
  • 检查DNS服务器是否为192.168.1.23

2021年8月27日

远程开会前1小时布置会议室

  1. 打开投影仪开关，对于三楼空中会议室，开关在灯光开关的左二带标记按钮
  2. 网线线槽-> USB网卡->接好电源的无线传屏器—HDMI—>电脑
  3. 在电脑上开传屏。
  4. 弄音响设备（如果有13人，蓝牙麦不够大声）
  • 转动钥匙开机
  • 用三叉线连音响、麦克风和电脑
  • 打开电脑的声音设置，从有线麦输入，排除笔记本内置的麦输入，输出到喇叭并测试三者
  1. 弄完声音才能开会议软件。海外会议用teams比钉钉更流畅，海外不适合视频会议只适合语音会议
  2. 发起会议->会议模式->开后会议->邀请群组->复制会议号在群里告诉参会人
  3. 把PPT开好。共享整个桌面，其他都不要选。按钮位置图示：| | |√| | | |
  4. 注意事项：用自己账号测试一下，开会前半小时再来看看，主持人不要结束会议

储运部断网问题修复

  1. 调整交换机D1的线序后，配置文件不会改变
  2. 网络良好时设备的“日志管理”的内容主要是：“Neighbor deleted on Port Ethernet1/0/…”
  3. 断网时，设备的“日志管理”的内容主要是：“Ethernet1/0/? link status is DOWN/UP.”

断网修复后善用win自带的“诊断”功能
